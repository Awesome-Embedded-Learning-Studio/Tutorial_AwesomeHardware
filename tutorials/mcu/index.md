# 单片机硬件

> 以 STM32F103（BluePill）为载体，讲单片机开发里**真正要懂的硬件**——不是教你调库，而是教你看懂手册、理解 GPIO 的物理结构、把"玄学问题"查到底。

## 这个版块在讲什么

市面上绝大多数 STM32 教程都在教「怎么调 HAL 库 / 标准库」，很少有人系统讲「怎么读 datasheet、GPIO 内部那几个 MOS 管在干嘛、电气特性表那一堆数字意味着什么」。从软件转硬件的人、做课设的学生，卡的就是这一层。

本版块补的就是这一层——**单片机开发里的硬件知识**。

## 载体：STM32F103C8T6（BluePill）

- **为什么是 F103**：Cortex-M3 内核、架构经典干净；中文资料密度最高（遇到任何坑都有人趟过）；最小系统板十几块钱，炸了不心疼。
- **不回避它的年龄**：F1 是 2007 年的老系列，GPIO 结构有反直觉的地方。这些都会讲清楚，并在末尾做系列对比（F1 → F4 → G0/G4），让 F103 成为出发点，而不是天花板。

## 资料与权威性

本版块的数值（电气参数、引脚定义、寄存器位等）**全部溯源到 ST 官方文档页码**，绝不凭记忆。主要参考：

- **DS5319 Rev 20**（2025-07）— STM32F103x8/B 中密度 datasheet
- **RM0008 Rev 21**（2021-02）— 参考手册
- **ES096 Rev 15**（2022）— 中密度 errata
- **PM0056**（2024-12）— Cortex-M3 编程手册
